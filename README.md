++ 不支持 12及以上版本的node
++ 可以使用nvm管理node版本
++++++++++++++++++
```
npm update -g @vue/cli // 升级命令
npm i -g @vue/cli
vue creat demo02cli
vue add vue-next
```
++ 也可以使用vite 构建工具 https://github.com/vitejs/vite
```
    $ npm init vite-app <project-name>
    $ cd <project-name>
    $ npm install
    $ npm run dev

    $ yarn create vite-app <project-name>
    $ cd <project-name>
    $ yarn
    $ yarn dev
```
